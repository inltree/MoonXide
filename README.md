> 移动端 GitHub 驱动 IDE —— 在 Android 上完成编码、AI 辅助、云端编译 APK 全流程。

## ✨ 核心能力（已完成）

- [x] **GitHub 集成**：Token 登录、多账号管理、仓库与文件树操作
- [x] **代码编辑器**：远程文件编辑、撤销/重做、搜索替换、脏状态标记
- [x] **AI 助手**：支持 OpenAI/Anthropic，可读写文件、调用 GitHub API
- [x] **云编译**：触发 GitHub Actions 构建 APK，自动获取产物与日志
- [x] **签名与发布**：Keystore 配置、Release 管理
- [x] **主题系统**：深浅色模式 + 多套配色 + 自定义背景

## 🚀 快速开始

```bash
flutter pub get
flutter run
```
> 确保已安装 Flutter SDK

## 📊 项目状态

✅ **当前已完成核心闭环**：  
登录 → 选择仓库 → 编辑 → AI 辅助 → 推送 → 云端编译 APK
