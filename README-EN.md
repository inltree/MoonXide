> Mobile GitHub-powered IDE — Complete coding, AI assistance, and cloud APK building on Android.

## ✨ Core Features (All Implemented)

- [x] **GitHub Integration** – Token login, multi-account management, repo & file tree operations
- [x] **Code Editor** – Remote file editing, undo/redo, search & replace, dirty state tracking
- [x] **AI Assistant** – Supports OpenAI/Anthropic, can read/write files, call GitHub API
- [x] **Cloud Build** – Trigger GitHub Actions to build APK, automatically fetch artifacts and logs
- [x] **Signing & Release** – Keystore configuration, Release management
- [x] **Theming** – Light/Dark mode + multiple color schemes + custom backgrounds

## 🚀 Quick Start

```bash
flutter pub get
flutter run
```
> Make sure Flutter SDK is installed

## 📊 Project Status

✅ **Core loop completed**:  
Login → Select repo → Edit → AI assistance → Push → Cloud build APK
